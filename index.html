datepicky.js

datepicky.js is a standalone javascript date picker, i.e. it has no library dependencies. While supporting the only date format that matters, ISO 8601 (YYYY-MM-DD), datepicky.js behavior can be easily customized for various use cases. It has been tested in IE6, IE7, IE8, IE9, FF5, Chrome 14, Safari 5, Safari iOS 4.3, and Opera 11.5.
